# Jameye Landing Page

Single-page Astro landing site for Jameye — the skill-ranked prediction tournament.

## Development

```bash
npm install
npm run dev
```

Site runs on http://localhost:4321.

## Build

```bash
npm run build
npm run preview
```

## Project structure

- `src/pages/index.astro` — single landing page
- `src/components/common/` — Header, Footer, Logo
- `src/content/data/home.json` — all landing copy
- `src/styles/global.css` — Tailwind theme + design tokens
- `src/config.ts` — site metadata
